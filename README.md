# SkyBlockX

> SkyBlock Plugin with SQLite3 Support

# UPDATE v1.0.5

- API Version: 3.19.1+dev
- Add Island Top
- Add Scoreboard

# Features
- Partner System
- Island Lock System
- The System of Seeing the Player on the Island
- Visiting System
- Throwing Players from the Island
- Island Erase System
- Creating an Island with a Generator System
